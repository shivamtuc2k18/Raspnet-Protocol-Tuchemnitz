uint32_t crc32b ( uint8_t *  message,
int  Psize 
)

to calculate the CRC value of payload.

Take the polynomial to a 32 bit variable.

Initially crc will be null.

Variable to check crc is calculated for all characters in the message.

When message has a character or Psize is not zero.

Perform XOR operation of crc with character in the message converted to 32 bit character.

Run the loop 8 times.

If MSB of crc is 1, right shift crc by 1 and XOR with polynomial.

If MSB of crc is 0, just right shift crc by 1.

Increment n by 1 to take the next character of the message.

Return the final computed 32 bit crc.
